Telegram Kalip Botu

Kurulum adimlari
  1. Telegram'da @BotFather acin → /newbot yazin.
  2. Bot adi verin (ornek: Fabrika Kalip).
  3. Kullanici adi xxxxxx_bot formatinda olmali (ornek: alfamold_bot).
  4. BotFather'in verdigi token'i asagiya yapistirin → Bot Ayarlarini Kaydet.
  5. Telegram aramada bot kullanici adinizi yazin (ornek @alfamold_bot) veya t.me/alfamold_bot acin.
  6. Bota /start basin (bir kez yeterli).
  7. Operatör Telegram ID ogrenmek icin: @userinfobot acin → Start → Id: 123456789 satirini okuyun.
  8. Operatör eklerken Seviye 1 (atama+üretim) veya Seviye 2 (sadece atama) seçin.
  9. QR formatlari: makine MACHINE:3 · kalip MOLD:042
